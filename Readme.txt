1)Extract Woof.rar
2)Open Extracted File from Android Studio Koala or Upper(Open Project)
3)run the app
4)then close the app and In Android Studio go to - Device Explorer/data/data/com.example.woof/databases
5)Right click on databases folder and select Upload
6)select the 2 files in Database folder and Upload (Woof & Woof-Journal)
7)Now you can run the app with Database
